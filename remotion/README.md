# Video landing · Perkins

Composiciones Remotion:

- `PerkinsHowItWorks` (1280×720, 15 s, 30 fps): intro + 3 pasos + cierre.
- `SantiagoMap` (720×720, 20 s, 30 fps): mapa de Santiago con pedidas y Perkins en ruta (hero landing).

## Comandos (desde la raíz del repo)

```bash
npm run remotion:studio        # previsualizar y editar
npm run remotion:render        # exporta a public/videos/perkins-como-funciona.mp4
npm run remotion:render:map    # exporta a public/videos/santiago-map.mp4
```

Tras cambiar una composición, vuelve a ejecutar el render correspondiente antes de deploy.
